Generalized section

Compute the section of e from public type œÑ to oblivious type œÑ'. e is assumed a body. Alternatively, we may assume e is locally closed, and open/close the expressions accordingly. That way we also need to record the used atoms in an extra arguments, similar to the lifting functions below.
Fixpoint gsec (œÑ œÑ' : expr) (e : expr) : option expr :=
  match œÑ, œÑ' with
  | <{ ùüô }>, <{ ùüô }> => Some <{ tape e }>
  | <{ ùîπ }>, <{ ~ùîπ }> => Some <{ tape (sùîπ e) }>
  | <{ œÑ1 * œÑ2 }>, <{ œÑ1' * œÑ2' }> =>
    e1 <- gsec œÑ1 œÑ1' <{ œÄ1 e }>;
    e2 <- gsec œÑ2 œÑ2' <{ œÄ2 e }>;
    Some <{ (e1, e2) }>
  | <{ œÑ1 + œÑ2 }>, <{ œÑ1' ~+ œÑ2' }> =>
    e1 <- gsec œÑ1 œÑ1' <{ bvar 0 }>;
    e2 <- gsec œÑ2 œÑ2' <{ bvar 0 }>;
    Some <{ tape (case e of ~inl<œÑ1' ~+ œÑ2'> e1 | ~inr<œÑ1' ~+ œÑ2'> e2) }>
  (* Outsource to predefined context *)
  | <{ gvar X }>, <{ X'@k }> =>
    '(s, _) <- Œî !! (X, X');
    Some <{ (gvar s) k e }>
  | _, _ => None
  end.

Generalized retraction

Compute the retraction of e from oblivious type œÑ' to public type œÑ.
Fixpoint gret (œÑ œÑ' : expr) (e : expr) : option expr :=
  match œÑ, œÑ' with
  (* Unlike section, retraction also allows the oblivious type œÑ' to contain
  non-oblivious components, as long as they match the corresponding components
  in œÑ. *)
  | <{ ùüô }>, <{ ùüô }> | <{ ùîπ }>, <{ ùîπ }> => Some e
  | <{ gvar X }>, <{ gvar Y }> =>
    if decide (X = Y) then Some e else None
  | <{ Œ†:{_}_, _ }>, <{ Œ†:{_}_, _ }> =>
    if decide (œÑ = œÑ') then Some e else None
  | <{ ùîπ }>, <{ ~ùîπ }> => Some <{ ~if tape e then true else false }>
  | <{ œÑ1 * œÑ2 }>, <{ œÑ1' * œÑ2' }> =>
    e1 <- gret œÑ1 œÑ1' <{ œÄ1 e }>;
    e2 <- gret œÑ2 œÑ2' <{ œÄ2 e }>;
    Some <{ (e1, e2) }>
  | <{ œÑ1 + œÑ2 }>, <{ œÑ1' + œÑ2' }> =>
    e1 <- gret œÑ1 œÑ1' <{ bvar 0 }>;
    e2 <- gret œÑ2 œÑ2' <{ bvar 0 }>;
    Some <{ case e of inl<œÑ1 + œÑ2> e1 | inr<œÑ1 + œÑ2> e2 }>
  | <{ œÑ1 + œÑ2 }>, <{ œÑ1' ~+ œÑ2' }> =>
    e1 <- gret œÑ1 œÑ1' <{ bvar 0 }>;
    e2 <- gret œÑ2 œÑ2' <{ bvar 0 }>;
    Some <{ ~case tape e of inl<œÑ1 + œÑ2> e1 | inr<œÑ1 + œÑ2> e2 }>
  (* Outsource to predefined context *)
  | <{ gvar X }>, <{ X'@k }> =>
    '(_, r) <- Œî !! (X, X');
    Some <{ (gvar r) k e }>
  | _, _ => None
  end.

Lifting

The core lifting function. All arguments are supposed to be locally closed, and the indices in œÑ' have been instantiated with free variables in xs. e of type œÑ is lifted to the oblivious counterpart of type œÑ'. xs keeps track of used free variables.
Fixpoint lift_core (œÑ œÑ' : expr) (xs : aset) (e : expr) : option expr :=
  match œÑ, œÑ' with
  | <{ Œ†:{l}œÑ1, œÑ2 }>, <{ Œ†:{l'}œÑ1', œÑ2' }> =>
    (* The function argument of e should have a leakage label of ‚ä§
    (LLeak), because it is applied to retraction. *)
    if l
    then let x := fresh xs in
         r <- gret œÑ1 œÑ1' <{ fvar x }>;
         (* œÑ and œÑ' are assumed non-dependent types, so œÑ2 and œÑ2'
         are locally closed. *)
         e' <- lift_core œÑ2 œÑ2' ({[x]} ‚à™ xs) <{ e r }>;
         Some <{ \:{l'}œÑ1' => ,(close x e') }>
    else None
  | _, _ => gsec œÑ œÑ' e
  end.

This function handles type index arguments and calls lift_core to do the heavy lifting (pun intended). All arguments are locally closed. It lifts e of type œÑ to the oblivious counterpart of type œÑ'. Unlike lift_core, œÑ' contains all the type index arguments from œÑs.
Fixpoint lift_ (œÑ œÑ' : expr) (œÑs : list expr) (xs : aset) (e : expr) : option expr :=
  match œÑs with
  | _::œÑs =>
    match œÑ' with
    | <{ Œ†:{_}œÑ1, œÑ2' }> =>
      let x := fresh xs in
      e' <- lift_ œÑ <{ œÑ2'^x }> œÑs ({[x]} ‚à™ xs) e;
      Some <{ \:œÑ1 => ,(close x e') }>
    | _ => None
    end
  | [] => lift_core œÑ œÑ' xs e
  end.

The top level lifting function. It lifts e of type œÑ to the oblivious counterpart of type œÑ', whose public indices are œÑs.
Definition lift (œÑ œÑ' : expr) (œÑs : list expr) (e : expr) : option expr :=
  lift_ œÑ œÑ' œÑs ‚àÖ e.

Well-formedness of lifting input

œÑ is not a dependent type (at top level). It may still take a higher-order function that is dependent type.
Fixpoint nodep (œÑ : expr) : Prop :=
  match œÑ with
  | <{ Œ†:{_}_, œÑ2 }> =>
    lc œÑ2 /\ nodep œÑ2
  | _ => True
  end.

lift_type_wf œÑs œÑ' means œÑ' is a valid lifted oblivious type with indices œÑs. First, œÑs should be the "prefix" of œÑ', i.e. œÑ' is of the form Œ†:œÑ1, Œ†:œÑ2, ..., œÑ where œÑs is œÑ1, œÑ2, .... Second, index types in œÑs are locally closed, i.e. there is no dependent type in indices. Third, the type with indices instantiated is non-dependent, i.e. œÑ should have the form œÑ1 -> œÑ2 -> .... They can of course depend on the indices though.
Inductive lift_type_wf : list expr -> expr -> Prop :=
| LTVNil œÑ' : nodep œÑ' -> lift_type_wf [] œÑ'
| LTVCons œÑk œÑs œÑ' L :
    lc œÑk ->
    (forall x, x ‚àâ L -> lift_type_wf œÑs <{ œÑ'^x }>) ->
    lift_type_wf (œÑk::œÑs) <{ Œ†:œÑk, œÑ' }>
.
